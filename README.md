# CEP Search - Python

Aplicação em Python que consome a API ViaCEP para buscar informações de um endereço a partir do CEP.

## 🚀 Funcionalidades
- Consulta de CEP
- Exibição de dados formatados
- Tratamento de erro para CEP inválido

## 🛠 Tecnologias
- Python
- Requests
- API REST (ViaCEP)

## ▶ Como executar
```bash
pip install requests
python main.py
